# Проблемы и улучшения

| Проблема | Последствие | Улучшение |
|---|---|---|
| Заявки поступают из разных каналов | Часть заявок теряется или обрабатывается с задержкой | Единая форма заявки в 1С |
| Данные проверяются вручную | Ошибки и неполные карточки заявок | Обязательные поля и автоматическая валидация |
| Статусы вводятся вручную | Невозможно корректно анализировать процесс | Справочник статусов |
| Нет контроля сроков | Просрочки замечаются поздно | SLA и автоматический контроль дедлайнов |
| Нет уведомлений | Ответственные не видят новые или просроченные заявки | Уведомления менеджеру и руководителю |
| Согласование проходит вне системы | Непрозрачная история решений | Согласование внутри 1С |
| Руководитель собирает отчет вручную | Потеря времени и риск ошибок | Управленческий отчет по заявкам |

## Ожидаемый эффект

- Сокращение ручных операций.
- Снижение количества неполных заявок.
- Повышение прозрачности процесса.
- Быстрый контроль просрочек.
- Единая аналитика по статусам, срокам и ответственным.
