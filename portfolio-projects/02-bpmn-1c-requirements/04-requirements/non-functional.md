# Non-Functional Requirements

| ID | Requirement | Acceptance Target |
|---|---|---|
| NFR-01 | Форма заявки должна быть понятна пользователям без дополнительного обучения | пользователь может создать заявку по подсказкам формы |
| NFR-02 | Отчет руководителя должен формироваться быстро | до 10 секунд при объеме до 10 000 заявок |
| NFR-03 | Изменение справочников должно быть ограничено правами | доступ только у администратора |
| NFR-04 | История статусов должна быть защищена от ручного удаления пользователем | изменение доступно только системе или администратору |
| NFR-05 | Интерфейс должен поддерживать ежедневную работу менеджера | ключевые поля доступны на первом экране формы |
