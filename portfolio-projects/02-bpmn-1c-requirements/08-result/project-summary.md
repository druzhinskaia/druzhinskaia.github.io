# Project Summary

## Case

Смоделированный коммерческий кейс по автоматизации обработки клиентских заявок в 1С для компании ООО "ТехСнаб".

## Business Problem

Заявки поступают из разных каналов, обрабатываются вручную и не имеют единого контроля сроков. Из-за этого возникают просрочки, неполные данные, разные статусы и ручная отчетность.

## Analyst Work

- описан бизнес-контекст и stakeholders;
- собраны пользовательские проблемы через интервью;
- описан AS-IS процесс и подготовлена BPMN-схема;
- сформирован scope проекта;
- подготовлены business и functional requirements;
- спроектирован TO-BE процесс;
- подготовлен UI mockup формы заявки;
- составлено техническое задание на доработку 1С;
- описаны acceptance criteria и тестовые сценарии.

## Deliverables

- AS-IS BPMN;
- TO-BE BPMN;
- Business Requirements;
- Functional Requirements;
- User Stories;
- Technical Specification;
- UI Mockup;
- Acceptance Criteria;
- Test Scenarios.

## Result

Подготовлен пакет аналитических артефактов, который можно передать команде 1С для оценки и реализации доработки.

## Skills

BPMN, draw.io, business analysis, process analysis, requirements management, user stories, technical specification, acceptance criteria, 1C.
