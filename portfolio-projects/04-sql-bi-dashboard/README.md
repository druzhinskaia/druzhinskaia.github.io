# SQL & BI Dashboard

## Описание

Проект показывает навыки SQL-анализа и подготовки BI-дашборда. Цель - построить небольшую модель данных, написать аналитические SQL-запросы и визуализировать ключевые показатели.

## Бизнес-ситуация

Руководителю нужен обзор продаж, клиентов, оплат и статусов заказов: выручка, количество заказов, средний чек, задолженность, топ-категории и динамика по месяцам.

## Что должно быть в проекте

- Синтетическая база данных.
- SQL-скрипты создания таблиц и загрузки данных.
- SQL-запросы для аналитики.
- BI-дашборд или макет дашборда.
- Скриншоты результата.
- Краткие выводы.

## Инструменты

SQL, SQLite/PostgreSQL, BI dashboard, reporting, data visualization.

## Структура

- `data` - исходные CSV или база данных.
- `sql` - DDL, загрузка данных и аналитические запросы.
- `dashboard` - файл BI-дашборда или описание макета.
- `docs` - описание модели данных и метрик.
- `result` - итоговые выгрузки и отчеты.
- `screenshots` - скриншоты дашборда и запросов.
