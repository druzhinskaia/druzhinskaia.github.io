# Portfolio Projects

Здесь собраны учебные портфолио-кейсы для резюме аналитика.

## Структура

- `01-data-quality-excel-reporting` - контроль качества данных из 1С и Excel-отчетность.
- `02-bpmn-1c-requirements` - описание бизнес-процесса, BPMN и требования к доработке 1С.
- `03-sales-feedback-python-analytics` - анализ продаж, заявок и клиентской обратной связи на Python.
- `04-sql-bi-dashboard` - SQL-запросы и BI-дашборд.
