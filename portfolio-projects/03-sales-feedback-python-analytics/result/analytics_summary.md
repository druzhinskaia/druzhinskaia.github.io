# Аналитическая записка

## Короткий вывод

За период с января по июнь 2026 года в выборке 4995 заказов на сумму 635 500 827 руб. Средняя оценка клиентов составила 4.19 из 5, NPS - 26, доля заказов с проблемной обратной связью - 29.0%.

Главная повторяющаяся причина негативной обратной связи - **срок поставки**. Наиболее рискованный продукт по доле проблемных заказов - **Редуктор R-8**, категория риска - **Редукторы**.

## Что видно по данным

- Выручка держится стабильно, но качество клиентского опыта отличается по продуктам.
- Проблемные обращения чаще связаны не с самим фактом продажи, а с ожиданиями клиента: срок, комплектация, документы и пояснение цены.
- При ответе быстрее 12 часов средняя оценка составляет 4.53; при ответе дольше 24 часов - 3.42.
- Продукты с высокой выручкой и высокой долей проблем требуют приоритета, потому что влияют и на деньги, и на повторные продажи.

## Рекомендации

1. Для `Редуктор R-8` разобрать причину `срок поставки` и добавить контрольный шаг перед передачей заказа клиенту.
2. Ввести контроль первого ответа по почтовому каналу.
3. Для `Компрессор А-120` подготовить шаблон ответа по причине `срок поставки` и обновить описание в коммерческом предложении.
4. Раз в неделю смотреть матрицу “доля проблем x маржинальность”, чтобы выбирать фокус улучшений.
